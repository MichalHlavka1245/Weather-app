# EN
Go to app/src/main/java/com/example/weather_app/get-data.kt
In get-data.kt on line 30 put your APi key . To get API key you must sign up in https://openweathermap.org/ . You must wait some time ,because api keys need time for being active .
In your IDE open the build.gradle.kts

# SK
Choď na app/src/main/java/com/example/weather_app/get-data.kt

Vo svojom IDE otvor build.gradle.kts
